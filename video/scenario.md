# Video Demo Scenario — QAHub AI Hackathon 2026

## Duration: ~10 minutes

---

### Scene 1: Introduction (1 min)
- Show the repository structure
- Explain the multi-agent architecture (diagram)
- "We built a multi-agent AI orchestrator that performs end-to-end QA engineering"

### Scene 2: Clean State (30 sec)
- Run `./scripts/cleanup.sh`
- Show that all AI-generated artifacts are removed
- Only infrastructure remains (Docker, orchestrator, configs)

### Scene 3: Start InvenTree (1 min)
- Run `./scripts/setup.sh`
- Show InvenTree starting up in Docker
- Open InvenTree in browser at http://localhost:8080
- Log in as admin, show the Parts module

### Scene 4: Run the Orchestrator (3 min)
- Run `./scripts/run-orchestrator.sh --verbose`
- Show Phase 1: Research Agent analyzing documentation
- Show Phase 2: Test Design agents generating test cases
- Show Phase 3: Code Generation agents writing Playwright code
- Show Phase 4: Review agent checking quality
- Highlight the AI logs being generated in `ai-logs/`

### Scene 5: Review Generated Artifacts (2 min)
- Open `test-cases/ui-manual-tests.md` — show test case quality
- Open `test-cases/api-manual-tests.md` — show API test cases
- Open `traceability-matrix.md` — show requirement-to-test mapping
- Open `risk-matrix.md` — show risk-based prioritization
- Open `automation/ui/pages/` — show POM pattern
- Open `automation/ui/helpers/self-healing.ts` — show self-healing selectors
- Open `automation/api/tests/schema-contract.spec.ts` — show contract testing

### Scene 6: Run Tests Live (2 min)
- Run `./scripts/run-tests.sh`
- Show API tests executing (80 tests passing)
- Show UI tests executing with browser visible (`--headed`)
- Show HTML test reports

### Scene 7: Differentiating Features (30 sec)
Quick recap slides:
1. Multi-agent orchestration (not just chat)
2. Requirements Traceability Matrix
3. Risk-based test prioritization
4. Self-healing selectors
5. Contract testing (JSON Schema validation)
6. CI/CD ready (GitHub Actions)
7. Full AI logging and cost optimization

### Scene 8: Closing (30 sec)
- "Every artifact you saw was generated by AI agents under human architectural direction"
- "Zero manual test writing — 100% agent-generated, human-reviewed"
- Show the GitHub repository

## Recording Tips
- Use OBS Studio or Loom for recording
- Terminal font size: 16pt minimum
- Split screen: terminal + browser
- Pause at key moments for readability
